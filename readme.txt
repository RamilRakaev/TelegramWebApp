Для запуска приложения необходимы следующие настройки:
1) Подключение к базе данных в файле appsettings.json

Для работы телеграм бота в режиме Updates необходимы:
1) Реквизиты для входа в Google Calendar: APIKey, Идентификатор календаря
2) Token телеграм бота
3) Имена разрешённых телеграм пользователей

Запуск телеграм бота в режиме Webhook
1) Добавить services.AddHostedService<ConfigureWebhookService>() в классе Startup
2) Добавить свойство HostAddress с URL-адресом сервера

Инструкция по настройке и запуску телеграм бота
1) На вкладке "Панель Администратора" нажать "Настройки"
2) Ввести значения и отправить
3) На вкладке "Панель Администратора" нажать "Добавить телеграм пользователей"
4) Ввести Username указанный в телеграм аккаунте без знака знака "@" и отправить
5) На вкладке "Панель Администратора" нажать "Включить прослушивание", в 
случае успеха вверху страницы отобразится сообщение "Телеграм бот запущен"
